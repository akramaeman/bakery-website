# Bakery Website 🍞🥐
Responsive bakery website frontend with modern UI built using HTML, CSS, and Bootstrap.



A modern and fully responsive bakery website frontend built using **HTML, CSS, and Bootstrap**.  
This project showcases a clean UI design for a bakery shop with a focus on responsiveness and user-friendly layout.

---

## 🌟 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern and attractive UI
- Homepage with bakery branding
- Product showcase section
- Navigation bar and footer
- Built using Bootstrap components for faster development

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## 📱 Responsive Design

This website is fully responsive and adapts to different screen sizes including:
- Mobile phones 📱  
- Tablets 📱  
- Desktop screens 💻
